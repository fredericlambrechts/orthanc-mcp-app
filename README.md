# DICOM MCP Feasibility

Feasibility study for a Claude MCP App that embeds a DICOMweb viewer (OHIF / Stone / Cornerstone3D) inline in Claude, loading studies from public DICOMweb sources such as Softneta DICOM Library and the Orthanc demo server. Target surface is Claude Cowork (and Claude.ai web). Key doc: [feasibility-study.md](./feasibility-study.md).
